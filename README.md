# My Portfolio

This is a personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the main application logic and pages.
- `src/components`: Contains reusable UI components like Navbar, Hero, About, Skills, Projects, Contact, and Footer.
- `public`: Contains static assets like images.

## Customization

You can customize the content of the portfolio by editing the components in the `src/components` directory.
- `Navbar.tsx`: Navigation links.
- `Hero.tsx`: Hero section with introduction.
- `About.tsx`: About me section.
- `Skills.tsx`: List of skills.
- `Projects.tsx`: List of projects.
- `Contact.tsx`: Contact information.
- `Footer.tsx`: Footer content.
